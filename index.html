Hybrid HGNC/HomoloGene Load

    Input

    • Input is HGNC and HomoloGen clusters from the database

    Logs

    • Process Log
    • Diagnostic Log

    Reports

    • Hybrid Cluster Report ## delimited
    • Columns:
      1. The connected component ID (arbitrary)
      2. All clusters in the connected component
      3. HGNC clusters in the connected component
      4. HomoloGene clusters in the connected componenet
      5. Secondary Source
      6. Conflict value (dev use only)
      7. Rule # from the Requirements wiki

    Output Files

    • MRK_Cluster.bcp
    • MRK_ClusterMember.bcp
